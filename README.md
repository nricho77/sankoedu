# Feuilles de Temps — Architecture & Backend

## Description
Application complète pour la gestion des feuilles de temps multi-site.

## Technologies
- Azure Static Web Apps
- Azure Functions
- Microsoft Graph API
- SharePoint Online
- Authentification Hybride (MSAL + JWT)

## Structure
```
/frontend
/api
```

## Installation Azure
1. Créer une Azure Static Web App
2. Déployer `/frontend`
3. Lier Function App et déployer `/api`
4. Ajouter variables d'environnement backend
5. Lancer le Setup initial
